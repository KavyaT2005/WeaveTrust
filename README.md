WeaveTrust is a web-based textile production management system designed for small-scale weaving businesses.

## Features
- Owner & Weaver role-based dashboards
- Order assignment and tracking
- Daily production updates
- Automatic order completion notifications via WhatsApp (Twilio)
- Clear production history and earnings tracking

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS
- Twilio WhatsApp API

## Purpose
Built to digitize and simplify traditional textile workflows used in family-run weaving businesses.
